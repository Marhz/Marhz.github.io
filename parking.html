Projets de gestion de parking d'entreprise

Projet simulant la gestion d'un parking d'entreprise, l'utilisateur doit d'inscrire, être validé par l'administrateur et peu demander l'attribution d'une place.

L'administrateur peut gérer et créer places, attributions et utilisateurs de l'application.



Langages et applications utilisées :

  • Html/Css

  • Php

  • Laravel 5.3(framework)

  • Mysql

  • Github

  • Asana

Compétences acquises :

  • Github et versioning de projet

  • Travail de groupe

  • Maitrise du framework Laravel 5
